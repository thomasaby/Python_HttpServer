PythonHttp

Python HTTP Server

HTTP/1.1 server built from scratch in Python using the standard socket and threading libraries. The primary goal of this project was to learn the fundamentals of network programming, the HTTP protocol, concurrency, and low-level server operations without relying on high-level frameworks like Flask or Django.
